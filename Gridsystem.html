Equal Width

Can be a whole grid that can be divided into multiple sections, all these sections are equaly divided depending on the number of boxes or grids you desire to have.


All Breakpoints

For grids are the same no matter the size, use the .col and .col-* classes. For other grids that are different sizes each you will use .col.


Stacked with Horizontal

Using a single set of .col-sm-* classes, this makes it to where if they are listed from top to bottom instead of left to right, using this you will be able to put them together horizontally.


Mix & Match

By mixing and matching you can make each grid a different size than the other without having to make them equal or the same class.


Row Columns

Use the responsive .row-cols-* classes to quickly set the number of columns that best render your content and layout. Whereas normal .col-* classes apply to the individual columns (e.g., .col-md-4), the row columns classes are set on the parent .row as a shortcut. With .row-cols-auto you can give the columns their natural width.
